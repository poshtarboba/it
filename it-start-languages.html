Мови програмування

Мова програмування — штучна мова, створена для опису команд, які мають виконати комп'ютери.

Мови програмування використовуються для створення програм, які контролюють поведінку машин, та для запису алгоритмів.

Алгоритм - набір інструкцій, які описують порядок дій виконавця, щоб досягти результату розв'язання задачі за скінченну кількість дій; система правил виконання дискретного процесу, яка досягає поставленої мети за скінченний час. Для візуалізації алгоритмів часто використовують блок-схеми.

Окрім мов програмування є ще мови опису та оформлення даних, їх не слід плутати з мовами програмування. До таких мов належать HTML, CSS, XML, JSON, LaTeX і багато інших.

Мови програмування можуть бути як відносно універсальними (застосовуються у різних сферах ІТ), так і вузькоспеціалізованими, заточеними під певні потреби.

Мови програмування можуть бути реалізовані як компільовані та інтерпретовані.

Компіляція - із написаної програми створюється файл (файли) з байт-кодом, цей код розуміє і швидко виконує процесор, але не розуміє людина, програміст. Під ОС Windows програми компілюються в EXE-файли (та деякі інші). При будь-якій зміні в коді програми цей код потрібно заново зкомпілювати.

Інтерпретація - виконання написаного коду програми крок за кроком без створення файлу з байт-кодом (перетворення у байт-код відбувається на льоту). При зміні коду програми ці зміни відбудуться при наступній інтерпритації. Інтерпритований код програми працює значно повільніше, ніж зкомпільований байт-код, але він гнучкіший і часто зручніший для розробників.

Деякі мови можуть і компілюватися, і інтерпритуватися. Код програм для інтерпритаторів часто називають скриптами.

Мови опису та оформлення (розмітки)

HTML (XHTML, HTML5, приклади) - мова розмітки, на якій написані веб-сторінки в інтернеті. В дужках - різні варіації мови HTML.

CSS (CSS3, SCSS, LESS, SASS, Stylus, приклади) - мова стилів, оформлення веб-сторінок: шрифти, колір, фон, рамки, відступи. В дужках представлені так звані препроцесорні мови стилів для веб-сторінок, дозволяють зручніше та швидше писати CSS-код.

XML (приклади) - мова, що описує дані, призначена для збереження в основному текстової інформації та передачі її через інтернет. Наразі потрохи відмирає.

JSON (приклади) - мова (формат даних), аналогічно формату XML зберігає дані та передає їх через інтернет. Витісняє XML завдяки своїй компактності і зручності використання.

SVG (приклади) - мова опису векторних зображень, схожа на HTML.

MarkDown (приклади) - проста і легка мова оформлення документів, використовується, наприклад, в проектах GitHub.

TeX / LaTeX (приклади) - мова оформлення наукових документів, підготовки їх до друку. Містить засоби для представлення різноманітних формул.

Є інші мови, менш розповсюджені та вузькоспеціалізовані.

Мови програмування

Assembler (приклади) - низькорівнева мова програмування, призначена для написання програм безпосередньо для процесора, пам'яті, різних чіпів, для написання драйверів до комп'ютерної переферії.

C, C++ (приклади) - універсальні мови програмування, застосовуються для написання операціїних систем, програм, додатків, ігор, драйверів, сервісних програм, можуть забезпечувати роботу сайтів на серверах.

Java, C# (приклади) - мови програмування для віртуальних машин, доволі універсальні. Застосовуються для написання програм, додатків, ігор. Java часто використовується на серверній частині веб-сайтів.

Віртуальна машина - програма, яка емулює комп'ютер. Під різні операційні системи пишуться окремі програми - віртуальні машини, які всередині працюють однаково і майже не залежать від заліза, на якому вони запускаються. Це дозволяє писати один варіант програми для різних операційних систем та пристроїв.

Клієнт і сервер (клієнт-серверні технології): програми, що поділені на 2 частини: одна частина запускається і працює на пристрої користувача, інша - на сервері в інтернеті. Приклади клієнт-серверних програм: сайти (клієнт - браузер користувача, сервер - база даних і програми на сервері), онлайн-ігри та безліч спеціалізованих програм, що зберігають дані централізовано.

Python (приклади) - скриптова (інтерпритована) мова програмування (може і компілюватися), на якій пишуть програми для штучного інтелекту, для роботи веб-сервера, для роботи з великими об'ємами тексту і т.п.

Basic, Pascal (приклади) - застарілі прості мови програмування, на яких було добре вивчати програмування. На базі Pascal було створено мову Object Pascal (середовище Delphi), на якій було досить просто створювати програми з гарним користувацьким інтерфейсом для ОС Windows.

Visual Basic (приклади) - оновлений Basic з ООП, використовується для написання програм під ОС Windows, а також широко застосовується для програмування у документах MS Office. Як і Python - може компілюватися або інтерпритуватися у вигляді скрипта.

JavaScript (TypeScript) (приклади) - мова програмування, що виконується на клієнтській частині веб-сайту (у браузері). Згодом JavaScript стало можливим запускати на серверній стороні (Node.js), а також його іноді застосовують у програмах керування кавоварками, пральними машинами, дронами, роботами-іграшками і т.п. техніки.

PHP (приклади) - застосовується на серверах веб-сайтів в основному для доступу до бази даних і формуванню веб-сторінок.

Lisp (приклади) - застосовується для програмування штучного інтелекту, лінгвістичних програм, вбудовується у інші програми як скриптова мова (наприклад, в AutoCad - програму для креслення).

Matlab (приклади) - математична мова програмування для математичних програм.

1С (приклади) - мова програмування, розроблена росіянами для різноманітних бухгалтерських, складських програм.

SQL (приклади) - не зовсім мова програмування, а мова запитів до баз даних.

Серед популярних мов є ще Go, R, Ruby, Perl, Haskel, Lua, Swift, Groovy...

Щороку з'являються нові мови програмування, більшість яких не набувають поширення або застосовуються лише у певних сферах.

Багато мов програмування залишилися в історії і зараз майже не використовуються.
